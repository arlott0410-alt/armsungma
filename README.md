# ArmSungMa Landing Page

Production-ready single-page landing page for ArmSungMa built with Vite, React, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy (Cloudflare Pages)

- **Build command:** `npm run build`
- **Output folder:** `dist`

## Notes

- Language toggle persists in `localStorage` (default Lao).
- Primary CTA: WhatsApp. Secondary: Telegram + Phone.
